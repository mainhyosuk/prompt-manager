## 설치 방법

1. 프로젝트 클론: `git clone https://github.com/mainhyosuk/prompt-manager.git`
2. 종속성 설치: 
   - 프론트엔드: `npm install`
3. 실행:
   - 프론트엔드: `npm run dev`
